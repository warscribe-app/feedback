# Warscribe Feedback

This is the place to report bugs, request features, and flag data issues for all Warscribe companion apps.

**The apps:**
- [AoS Tracker](https://aos.warscribe.app) — Age of Sigmar 4th Edition
- [40k Tracker](https://40k.warscribe.app) — Warhammer 40,000 10th Edition
- [Spearhead Tracker](https://spearhead.warscribe.app) — Spearhead
- [The Old World Tracker](https://tow.warscribe.app) — Warhammer: The Old World
- [Path to Glory](https://ptg.warscribe.app) — AoS Path to Glory Campaigns

## How to report a bug

Click **New Issue** and pick the right template:

- **Bug Report** — something's broken or behaving unexpectedly
- **Data Issue** — wrong points, missing units, incorrect ability text, etc.
- **Feature Request** — ideas for new features or improvements

Include as much detail as you can — which app, what you were doing, and what went wrong. Screenshots and army list text are really helpful.

## What happens next

Every report gets read. Bug fixes and data corrections typically ship within a day or two. Feature requests get tracked and prioritised based on how many people ask for the same thing.
